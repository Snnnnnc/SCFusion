# BDF文件分析指标说明

## 1. 采样点唯一值数量（Unique Values Count）

### 定义
统计前10000个采样点中，有多少个**不同的数值**。

### 示例
```python
# data.bdf的前10个采样点：
[18.11, 18.11, 18.11, 18.11, 18.11, 18.11, 18.11, 18.11, 18.11, 18.11]
# 唯一值数量 = 1（只有一个不同的值：18.11）

# wl.bdf的前10个采样点：
[18.11, 18.98, 19.58, 19.91, 20.00, 19.87, 19.57, 19.13, 18.62, 18.08]
# 唯一值数量 = 10（10个不同的值）
```

### 意义

#### 对于100Hz原始数据上采样到1000Hz的情况：
- **原始100Hz数据**：每秒100个采样点，每个点都是新数据
- **上采样到1000Hz**：通常采用**重复采样**（hold）或**零阶保持**方法
  - 方法：将每个100Hz的采样点重复10次
  - 结果：每10个连续的采样点**完全相同**
  
#### 数据特征：
- **data.bdf**: 唯一值数量 = 244（前10000个点）
  - 说明：大约每41个点才有一个新值（10000/244 ≈ 41）
  - 特征：**有明显的重复模式**，强烈暗示是100Hz数据被重复采样到1000Hz

- **wl.bdf**: 唯一值数量 = 9920（前10000个点）
  - 说明：几乎每个点都是新值（10000/9920 ≈ 1）
  - 特征：**数据连续变化**，可能是：
    1. 真正的1000Hz采样数据
    2. 或者是100Hz数据经过**插值**（如线性插值）上采样到1000Hz

### 判断标准
- **唯一值数量 << 采样点数**（如244/10000 = 2.4%）→ 明显的重复采样（上采样）
- **唯一值数量 ≈ 采样点数**（如9920/10000 = 99.2%）→ 数据连续，无重复采样特征

## 2. 每10个点重复组数（Repeated Segments Count）

### 定义
检查每10个连续的采样点是否**完全相同**，统计有多少组满足这个条件。

### 检查方法
```python
# 将数据分成每10个点一组
第0组: 索引 0-9   → [18.11, 18.11, 18.11, ..., 18.11]  # 全部相同？
第1组: 索引 10-19 → [18.15, 18.15, 18.15, ..., 18.15]  # 全部相同？
第2组: 索引 20-29 → [18.20, 18.20, 18.20, ..., 18.20]  # 全部相同？
...
第99组: 索引 990-999 → [...]
```

### 意义

#### 对于100Hz → 1000Hz的上采样（零阶保持）：
- **原始数据**（100Hz）：
  ```
  时间:  0.00s   0.01s   0.02s   0.03s  ...
  值:    18.11   18.15   18.20   18.25  ...
  ```

- **上采样后**（1000Hz，零阶保持）：
  ```
  时间:  0.000s  0.001s  0.002s  ...  0.009s  0.010s  0.011s  ...
  值:    18.11   18.11   18.11   ...  18.11   18.15   18.15   ...
         └───────── 重复10次 ─────────┘ └───────── 重复10次 ─────────┘
  ```

- **结果**：每10个连续的采样点**完全相同**

#### 数据分析结果：

**data.bdf:**
- 每10个点重复的组数：**100/100**（全部100组都重复）
- 判断：**确认是100Hz数据通过零阶保持上采样到1000Hz**

**wl.bdf:**
- 每10个点重复的组数：**0/100**（没有一组完全重复）
- 判断：**不是简单的重复采样**，可能是：
  1. 真正的1000Hz采样
  2. 或者100Hz数据通过**插值**上采样（线性插值、样条插值等）

## 3. 两个指标的联合判断

### 情况A：明显的上采样（零阶保持）
- **唯一值数量**: 很小（如244/10000 = 2.4%）
- **每10个点重复**: 大量组数（如100/100 = 100%）
- **结论**: ✅ **100Hz原始数据，通过零阶保持上采样到1000Hz**
- **数据质量**: ⚠️ 包含大量重复值，信息密度低

### 情况B：插值上采样或真实1000Hz采样
- **唯一值数量**: 很大（如9920/10000 = 99.2%）
- **每10个点重复**: 很少或没有（如0/100 = 0%）
- **结论**: ✅ **可能是100Hz插值上采样，或者是真实1000Hz采样**
- **数据质量**: ✅ 数据连续变化，信息密度高
